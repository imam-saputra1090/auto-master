🛠️ Panduan & Salin Apps Script Database (Untuk Rekan Guru)

Ingin menggunakan game ini untuk kelas Anda sendiri di sekolah? Anda bisa menduplikasi database spreadsheet secara mandiri dengan langkah mudah berikut:

  1. Buat Spreadsheet Baru: Buat sebuah spreadsheet kosong baru di Google Drive Anda.
  2. Buka Google Apps Script: Pada menu atas spreadsheet, klik Ekstensi > Apps Script.
  3. Tempel Kode Backend: Hapus seluruh kode bawaan di editor Apps Script, salin kode Apps Script di bawah ini secara penuh dan tempelkan.
  4. Atur Password Guru Anda: Sesuaikan konfigurasi password dasbor Anda pada variabel TEACHER_SECRET di baris ke-30 (ganti sesuai keinginan Anda).
  5. Jalankan Fungsi Setup: Pilih fungsi SETUP pada menu dropdown bagian atas editor, kemudian klik tombol Run / Jalankan. Fungsi ini akan membuat lembar kerja users, scores, dan sessions secara otomatis di Spreadsheet Anda.
  6. Terapkan Aplikasi Web: Klik tombol Terapkan (Deploy) > Terapkan Baru (New Deployment). Pilih tipe **Aplikasi Web (Web App)**. Konfigurasikan:
    • Jalankan sebagai (Execute as): Saya / Me (akun Google Anda)
    • Siapa yang memiliki akses (Who has access): Siapa saja / Anyone
  7. Hubungkan ke Dasbor: Salin URL Aplikasi Web yang muncul setelah penerapan, masukkan pada kolom Konfigurasi Google Apps Script URL API di atas dasbor ini, lalu klik **Simpan 💾**.
